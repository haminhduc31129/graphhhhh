# 📊 Tổng Hợp Kiến Thức & Bài Tập Lý Thuyết Đồ Thị (Graph Theory)

Chào mừng bạn đến với dự án của **Nhóm 6**! Đây là kho lưu trữ (repository) tổng hợp các kiến thức nền tảng, lý thuyết chuyên sâu và các bài tập thực hành về **Lý Thuyết Đồ Thị** (Graph Theory). 

Các nội dung trong đây đều mang tính chất tham khảo, được nhóm chắt lọc và tổng hợp một cách khoa học từ nhiều nguồn học thuật uy tín nhằm mục đích học tập và nghiên cứu.

## 📂 Cấu Trúc Thư Mục

Dự án được tổ chức gọn gàng để dễ dàng tra cứu:

- 📁 **`lythuyet/`**: Chứa các tài liệu lý thuyết chi tiết về đồ thị. Các tài liệu này được soạn thảo chủ yếu bằng **LaTeX** nhằm đảm bảo việc trình bày các định lý, công thức toán học một cách chuẩn xác và chuyên nghiệp nhất.
- 📁 **`PROBLEM/`**: Thư mục chứa các bài tập, thuật toán và các bài toán đồ thị thực tế để người học có thể thực hành áp dụng lý thuyết.
- 📄 **`main.cpp`** & **`mai.py`**: Mã nguồn (Source code) cài đặt các thuật toán minh họa trên đồ thị (VD: duyệt đồ thị BFS/DFS, tìm đường đi ngắn nhất, v.v.) bằng ngôn ngữ C++ và Python.

## 🛠 Công Nghệ & Ngôn Ngữ Sử Dụng
- **LaTeX (~99.8%)**: Công cụ chính dùng để biên soạn tài liệu học thuật.
- **C++**: Ngôn ngữ lập trình được sử dụng để tối ưu hóa hiệu năng các thuật toán đồ thị.
- **Python**: Hỗ trợ việc cài đặt thuật toán nhanh và dễ hiểu.

## 🚀 Hướng Dẫn Sử Dụng

1. **Đọc tài liệu lý thuyết:** Mở thư mục `lythuyet`. Để xem cấu trúc đẹp nhất của các file `.tex`, bạn nên sử dụng các trình biên dịch LaTeX (như TeXstudio, VS Code extension) hoặc upload lên [Overleaf](https://www.overleaf.com/) để xuất ra file PDF.
2. **Thực thi mã nguồn (Code):** Bạn có thể tham khảo cách cài đặt thuật toán ở các file `main.cpp` hoặc `mai.py`.
   - *Biên dịch & Chạy C++:* ```bash
     g++ main.cpp -o main && ./main
     ```
   - *Chạy Python:* ```bash
     python mai.py
     ```
