\section{Giới thiệu}

\begin{frame}{Giới thiệu}

\begin{block}{Khái niệm}
Đồ thị $G = (V, E)$ là một cấu trúc rời rạc gồm tập các đỉnh $V$ và tập các cạnh $E$.
\end{block}

\begin{alertblock}{Thành phần}
\begin{itemize}
\item Đỉnh (Vertex)
\item Cạnh (Edge)
\item $deg(v)$ là bậc
\item $\sum deg(v) = 2|E|$
\end{itemize}
\end{alertblock}

\end{frame}

%--------------------------------
\begin{frame}{Ví dụ đồ thị}

\column{0.55\textwidth}

\begin{itemize}

\end{itemize}

\end{columns}

\end{frame}
